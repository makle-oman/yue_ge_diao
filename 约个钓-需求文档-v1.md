# 约个钓 需求文档 v1.0

> **版本**：v1.0（免费版 MVP）
> **日期**：2026-05-18
> **技术栈**：uniapp + Vue3，主战场微信小程序
> **目标**：个人开发者 4-8 周内可完成的 MVP

---

## 一、产品概述

### 1.1 一句话定位
基于地理位置的钓友社区小程序，让钓鱼爱好者**找到好钓点、约到好搭子、记录每次鱼获**。

> **产品名：约个钓** —— 蹭"约个饭/约个X"的口语化命名，自带社交动作暗示。

### 1.2 核心价值主张
- **解决"去哪钓"**：UGC 钓点地图 + 实时鱼情
- **解决"和谁钓"**：附近钓友 + 组队约钓
- **解决"留念"**：鱼获打卡 + 朋友圈裂变

### 1.3 V1 边界（重要）
- ✅ **做什么**：UGC 内容生态、社交关系链、基础工具
- ❌ **不做什么**：付费功能、电商、广告、商家入驻、直播

## 四、详细功能需求

### F1. 用户系统（P0）

**功能描述**
微信一键登录，自动获取头像和昵称，引导完善钓鱼相关资料。

**需求点**
- 登录方式：微信授权登录（getUserProfile）
- 首次登录引导填写：
  - 钓龄（1 年以内 / 1-3 年 / 3-5 年 / 5 年以上）
  - 主玩方式（多选：野钓 / 路亚 / 黑坑 / 海钓 / 冰钓）
  - 常驻城市（自动定位 + 可修改）
- 用户字段：
  - 头像、昵称、性别、钓龄、玩法标签、城市
  - 注册时间、最后活跃时间
- 隐私设置：
  - 是否允许被附近钓友看到
  - 是否公开鱼获位置

**验收标准**
- 3 秒内完成登录
- 未完善资料的用户在发布内容前必须先完善

---

### F2. 钓点地图（P0 - 核心）

**功能描述**
打开即看到的核心页面，地图模式展示附近所有 UGC 钓点。

**需求点**

**2.1 地图展示**
- 默认定位到用户当前位置
- 显示半径：可切换 5km / 20km / 50km / 100km
- 钓点 marker 样式按类型区分：
  - 🌊 野钓点（河流/湖泊/水库）
  - 🎣 黑坑（v1 仅展示不做交易）
  - ⚓ 海钓码头
- 聚合显示（地图缩小时合并附近点）
- 支持搜索：城市、钓点名称

**2.2 钓点信息展示**
点击 marker 弹出卡片：
- 钓点名称、类型、距离
- 主要鱼种（标签）
- 最近鱼获数（近 7 天 / 近 30 天）
- 最近一次有人去的时间
- 平均评分（5 星制）
- "查看详情"按钮 → 跳钓点详情页

**2.3 上传新钓点（UGC 关键）**
- 入口：地图右下角"+"按钮
- 必填字段：
  - 钓点名称
  - 位置（自动获取当前定位，可拖动调整）
  - 类型（野钓 / 黑坑 / 海钓码头）
  - 水域类型（河 / 湖 / 水库 / 池塘 / 海岸 / 海上）
- 选填字段：
  - 主要鱼种（多选标签）
  - 是否收费、停车情况、是否有厕所
  - 现场照片（最多 9 张）
  - 文字描述
- **防作弊**：必须现场定位（位置精度 < 50m），不允许虚拟定位

**验收标准**
- 地图加载 < 2 秒
- 钓点 marker 渲染顺畅，500 个点不卡
- 上传一个新钓点流程 < 1 分钟

### F3. 鱼获打卡（P0 - 裂变关键）

**功能描述**
钓友最重要的"晒"场景，是产品最强的传播路径。

**需求点**

**3.1 发布鱼获**
- 入口：底部 Tab 中间凸起的"+"
- 必填：
  - 至少一张照片
  - 鱼种（输入或从标签选）
  - 钓点（从地图选 或 关联已有钓点）
- 选填：
  - 重量、长度
  - 使用饵料 / 钓法
  - 天气 / 气压（自动填充）
  - 文字描述
- 隐私选项：
  - 是否公开钓点位置（保护钓友的"秘密点"）
  - 是否允许评论

**3.2 鱼获详情页**
- 大图浏览
- 钓点信息（除非用户选择隐藏）
- 当时天气、气压、风向
- 点赞、评论、收藏
- 分享到微信好友 / 朋友圈（生成精美卡片）

**3.3 朋友圈分享卡片**
- 自动生成卡片：背景图 + 鱼种 + 重量 + 钓友圈 logo
- 这是 V1 最重要的获客来源，**设计必须精致**

**验收标准**
- 发布流程 < 90 秒
- 分享卡片在朋友圈展示效果好（关键指标：点击率）

### F5. 天气 + 宜钓指数（P0）

**功能描述**
钓鱼佬出发前的"占卜"工具，强日活驱动。

**需求点**

**5.1 数据来源**
- 接入和风天气 API 免费版（每日 1000 次够个人项目用）
- 数据项：温度、气压、风向风速、降水、能见度、月相

**5.2 宜钓指数算法（v1 简版）**
```
基础分 60，加减项：
+ 气压稳定 / 上升  +10
+ 微风（3-5 级）   +10
+ 阴天 / 多云      +5
+ 春秋季           +5
- 暴雨 / 大风      -30
- 气压骤降         -15
- 极端高温/低温    -10
```
最终输出 0-100 的"宜钓指数"，并给一句话解读。

**5.3 展示位置**
- 首页地图顶部横幅
- 个人主页快捷入口
- 订阅消息推送（用户授权后）

**验收标准**
- 算法逻辑可调（独立配置文件，方便迭代）

### F6. 附近钓友列表（P1）

**功能描述**
LBS 列出附近活跃钓友。

**需求点**
- 列表/地图两种视图
- 排序：距离 / 活跃度
- 显示：头像、昵称、距离、最近一次活跃、主玩方式标签
- 点击 → 个人主页
- 隐私：用户可关闭"被发现"

---

### F7. 组队约钓（P1）

**功能描述**
钓友 LBS 组队功能。

**需求点**

**7.1 发起组队**
- 字段：
  - 钓点（关联已有 或 自定义位置）
  - 时间（具体日期 + 时间段）
  - 目标鱼种
  - 限制人数（1-10）
  - AA / 请客 / 各自承担
  - 是否需要拼车
  - 备注
- 发布后默认公开到"附近"列表

**7.2 报名 & 管理**
- 其他用户点击"报名"，发起人审核
- 报名通过后双方可加微信群（小程序内不做 IM，降低复杂度）
- 状态流转：招募中 → 已满员 → 进行中 → 已结束

**7.3 防鸽机制（重要）**
- 爽约：发起人可在结束后标记"未到场"
- 系统记录"放鸽次数"，超过阈值降低优先级

## 七、技术栈

### 7.1 前端
- **框架**：uniapp + Vue 3 + TypeScript
- **状态管理**：Pinia
- **UI 组件**：uView Plus 或 uni-ui（按需）
- **地图**：腾讯位置服务（微信小程序首选）
- **图片处理**：uniapp 内置 + 云存储

### 7.2 后端（二选一）
**方案 A：uniCloud（推荐个人开发）**
- 优势：和 uniapp 深度集成、免运维、有免费额度
- 劣势：被锁死在 DCloud 生态

**方案 B：独立后端（Node.js + Express/Nest）**
- 优势：灵活、可平滑迁移
- 劣势：需要自己买服务器和搞运维

### 7.3 第三方服务
| 服务 | 用途 | 成本 |
|---|---|---|
| 腾讯位置服务 | 地图、定位、POI | 免费额度足够 |
| 和风天气 | 天气数据 | 免费版 1000/天 |
| 百度 AI 开放平台 | 鱼种识别（P2） | 0.001-0.01/次 |
| 微信订阅消息 | 推送 | 免费 |

## 八、非功能需求

### 8.1 性能
- 首屏加载 < 2 秒
- 地图渲染 500 个 marker 不卡顿
- 图片上传走 CDN

### 8.2 安全 & 合规
- 用户位置数据加密存储
- UGC 内容审核（接入微信内容安全 API）
- 敏感水域（水库、禁渔区）主动屏蔽 + 免责声明
- 用户可一键删除自己所有数据

### 8.3 体验
- 离线可查看已缓存钓点
- 弱网友好（图片渐进加载）
- 完整的空状态/错误状态设计
