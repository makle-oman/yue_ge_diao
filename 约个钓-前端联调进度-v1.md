## 关键约定

| 约定 | 内容 |
|---|---|
| 响应信封 | `{ code: 200, msg: "ok", data, traceId }` |
| 成功码 | `code === 200` 才算业务成功 |
| 鉴权 | 全局 `JwtAuthGuard`；公开接口需 `@Public()` |
| Trace | 前端发 `X-Trace-Id`，后端响应回带 `traceId` |
| baseUrl | 开发：`http://localhost:3000/api` |
| 上传 | `multipart/form-data` 字段名 `file`；jpeg/png/webp；10MB；返回 `width`/`height` |
