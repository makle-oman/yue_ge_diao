## 0. 设计风格总定调

### 0.1 风格关键词
**自然清新 + 工具感** —— 湖水蓝绿主色 + 数据化呈现

### 0.2 色彩系统（建议初版）
```
主色 Primary    : #2D8F87  湖水墨绿（沉稳、不刺眼）
辅色 Secondary  : #5BA9C4  天空蓝（活力）
强调 Accent     : #F5A623  夕阳橙（CTA 按钮、关键数据）
成功色          : #4CAF50  鱼获成功绿
警告色          : #FF6B6B  气压突变红
背景 Background : #F7F9FA  浅灰白
文字主色        : #1A2B33  墨色
文字次色        : #6B7B85  灰
分割线          : #E5EAED
```

### 0.3 字体 & 圆角
- 字体：**苹方 / PingFang SC**（小程序默认即可）
- 圆角：卡片 12rpx、按钮 8rpx、头像圆形
- 阴影：`0 2rpx 12rpx rgba(0,0,0,0.06)` 轻投影

### 0.4 视觉气质
- ❌ 不要：卡通鱼 emoji 满天飞、Q 版、马卡龙色
- ❌ 不要：纯深色背景（钓鱼佬阳光下看不清）
- ✅ 要：留白多、信息密度中等、关键数据加粗放大
- ✅ 要：照片为主视觉，UI 让位给内容

## 1. TabBar 总体结构

### 1.1 五个 Tab 布局

```
┌─────────────────────────────────────────────┐
│                                             │
│              页面内容区                      │
│                                             │
├──────┬──────┬──────┬──────┬──────────────┤
│ 🗺️   │ 🐟   │  ➕  │ 💬   │ 👤            │
│ 钓点 │ 社区 │      │ 消息 │ 我的          │
└──────┴──────┴───↑──┴──────┴──────────────┘
                  └─ 中间凸起按钮，比其他 tab 大 30%
```

### 1.2 TabBar 元素规范
- 高度：100rpx（不含安全区）
- 图标尺寸：48rpx × 48rpx
- 中间"+"：80rpx × 80rpx 圆形，上浮 20rpx，主色背景 + 白色加号
- 选中态：图标变主色 + 文字加粗
- 默认态：图标 #999、文字 #999

### 1.3 中间"+"点击交互
**不跳页**，从底部弹出 ActionSheet（半屏抽屉）：
```
┌─────────────────────────┐
│         发布            │
├─────────────────────────┤
│  🐟  发布鱼获  >        │
│  📍  上报钓点  >        │
│  👥  发起组队  >        │
├─────────────────────────┤
│        取消             │
└─────────────────────────┘
```

## 2. Tab 1：钓点（地图首页）★ 最核心

### 2.1 页面定位
打开 App 第一眼，**地图 + 附近钓点 + 宜钓指数**，让用户立刻找到"今天去哪钓"。

### 2.2 内容元素清单

**顶部区（从上到下）**
1. 沉浸式状态栏
2. 顶部条：
   - 左：城市切换 `南京 ▼`
   - 中：搜索框 `搜索钓点/地址/鱼种`
   - 右：消息铃铛图标（带红点未读数）
3. 筛选 chips（横向滚动）：
   - `全部` `野钓` `黑坑` `海钓` `5km内` `鲫鱼` `鲈鱼` `+更多`

**主体区**
4. 全屏地图（占据屏幕 80%）
5. 钓点 Marker 三种样式：
   - 🌊 野钓（蓝色水滴 icon）
   - 🎣 黑坑（橙色鱼竿 icon）
   - ⚓ 海钓（深蓝色锚 icon）
6. 当前位置：蓝色光点 + 涟漪动画
7. 聚合 marker：圆形 + 数字（地图缩小时）

**浮层**
8. 右上角：宜钓指数胶囊浮卡
   - 圆形进度环 + 数字（如 `82`）
   - 下方一行 `今日宜钓` + 表情（☀️ 🐟）
   - 点击展开详细天气
9. 右下角悬浮按钮组（垂直排列）：
   - 上：📍 定位按钮（白底）
   - 下：➕ 上报钓点（主色）

**底部上拉抽屉**
10. 抽屉吸底显示：
    - 标题栏 `附近钓点 · 共 23 个` + 排序按钮 `距离 ▼`
    - 横向滚动卡片（露出 1.5 张）：
      - 缩略图（钓点封面）
      - 钓点名 `燕子矶江边`
      - 距离 `2.3km`
      - 鱼种标签 `鲫鱼 鲤鱼 鳊鱼`
      - 评分 `⭐ 4.5（12 人评价）`
      - 最近鱼情 `今天有人来 · 钓获 3 条`

### 2.3 空状态设计
- 周围 50km 无钓点时：
  - 插画 + 文案 `这片水域还没有钓点`
  - CTA `成为第一个上报钓点的人 ➕`

### 2.4 ★ AI 出图提示词

```
设计一个微信小程序首页地图界面，钓鱼社区类 App，主用户为 30-50 岁中年男性钓鱼爱好者。

【整体风格】
- 自然清新 + 工具化数据呈现
- 主色：湖水墨绿 #2D8F87
- 辅色：天空蓝 #5BA9C4，夕阳橙 #F5A623（用于 CTA）
- 浅色背景，整体明亮通透，不要深色模式

【布局结构】从上到下：
1. 顶部沉浸式状态栏
2. 搜索栏：左侧城市选择"南京 ▼"，中间圆角搜索框"搜索钓点/地址"，右侧消息铃铛带红点
3. 筛选标签栏：横向滚动 chips「全部 / 野钓 / 黑坑 / 海钓 / 5km内 / 鲫鱼 / 鲈鱼」
4. 主体全屏地图（占 70% 屏幕）：
   - 显示中国某城市江边地图
   - 多个钓点 marker：蓝色水滴图标（野钓）、橙色鱼竿图标（黑坑）
   - 中央有蓝色光点表示用户当前位置，带涟漪动画
5. 地图右上角浮层：圆形进度环显示数字"82"，下方文字"今日宜钓"+ 太阳鱼图标 emoji
6. 地图右下角浮动按钮：上方白色定位按钮，下方墨绿色加号"上报钓点"
7. 底部上拉抽屉：标题"附近钓点 · 共 23 个" + 右侧"距离 ▼"，下方横向滚动卡片，露出 1.5 张：
   - 每张卡片：左侧缩略图，右侧钓点名"燕子矶江边"、距离"2.3km"、鱼种标签、评分 4.5 星
8. 底部 TabBar 5 个 icon：钓点（选中态-墨绿色）、社区、中间凸起的"+"圆形按钮、消息（带红点）、我的

【风格细节】
- 圆角 12rpx，卡片轻投影
- 字体苹方
- 数据加粗（如评分数字、距离）
- 留白充足，信息密度中等
```

## 3. Tab 2：社区（Feed 流）

### 3.1 页面定位
**鱼获瀑布流**，刷别人钓的鱼，是产品社区氛围核心。

### 3.2 内容元素清单

**顶部区**
1. 沉浸式状态栏
2. 顶部 Tab 切换（居中）：
   - `推荐` / `附近` / `关注`
3. 右上角搜索图标

**主体区（双列瀑布流）**
每张卡片元素：
4. 卡片顶部：
   - 头像（圆形）+ 昵称 + 钓龄标签
   - 右上角：距离 `2.1km`
5. 主图：鱼获照片（不等高瀑布流）
   - 鱼种角标：左下角胶囊 `鲫鱼 · 1.2斤`
6. 卡片底部：
   - 钓点（如果公开）：📍 `燕子矶江边`
   - 文案预览 1 行
   - 底部互动栏：❤️ 32 · 💬 8 · 🔗

### 3.3 空状态
- 关注 Tab 空：`还没关注钓友，去看看附近钓友 →`

### 3.4 ★ AI 出图提示词

```
设计一个微信小程序的钓鱼社区 Feed 流页面，类似小红书但聚焦钓鱼内容。

【整体风格】
- 浅色背景 #F7F9FA
- 主色湖水墨绿 #2D8F87，强调色夕阳橙 #F5A623
- 内容为主，UI 极简，让鱼获照片成为视觉焦点

【布局结构】
1. 顶部沉浸式状态栏
2. 顶部居中 Tab 切换：「推荐」「附近」「关注」三个选项，当前选中"推荐"显示墨绿色加粗+下划线
3. 顶部右上角搜索图标
4. 主体双列瀑布流卡片，每张卡片：
   - 顶部一行：圆形头像 + 用户昵称"老王钓鱼" + 钓龄标签"5年" + 右上角距离"2.1km"
   - 中间大图：钓鱼成果照片（鲫鱼、鲤鱼、路亚翘嘴等），不等高展示
   - 图片左下角胶囊标签：黑色半透明背景白字"鲫鱼·1.2斤"
   - 图片下方：📍 钓点名"燕子矶江边"（灰色小字）
   - 一行文案预览"今天气压回升，鱼口很活，半天三条板鲫…"
   - 底部一行：爱心 32 · 评论气泡 8 · 分享图标
5. 底部 TabBar：5 个 tab，社区图标选中态墨绿色

【内容示例】至少展示 6 张卡片，混合不同鱼种和不同场景（野钓、路亚、海钓）

【风格细节】
- 卡片圆角 16rpx，轻投影
- 文字层级清晰：用户名 14pt，鱼种标签 12pt 加粗，文案 13pt 灰
- 整体氛围像小红书+垂直钓鱼
```

## 5. Tab 4：消息

### 5.1 页面定位
聚合所有互动通知，**v1 不做即时聊天**（组队后跳微信群）。

### 5.2 内容元素清单

**顶部区**
1. 状态栏
2. 标题栏：左标题 `消息`，右"全部已读"文字按钮

**功能入口区（4 个 grid，2x2 或横向）**
3. 四个圆形大图标 + 数字徽章：
   - 💬 评论与@（红点 5）
   - ❤️ 收到的赞（红点 12）
   - 👥 组队通知（红点 2）
   - 📢 系统消息

4. "最近互动"列表（类似微信对话列表，但只是聚合通知）：
   - 头像 + 用户名 + 操作 + 时间
   - 示例：`@老王 赞了你的鱼获 · 5 分钟前`

### 5.3 空状态
- 整体空：插画 + `还没有消息哦，去社区看看吧`

### 5.4 ★ AI 出图提示词

## 7. 二级页面（重要）

以下二级页面虽然不在 TabBar，但是核心流程，**设计图也要画**。

### 7.1 钓点详情页

**元素清单**
- 顶部图片轮播（钓点环境照）
- 钓点基础信息卡：名称、类型、距离、导航按钮
- 实用信息：是否收费、停车、厕所、最佳时段
- 主要鱼种标签
- 宜钓指数 + 天气
- 历史鱼获瀑布流（关联此钓点的鱼获）
- 用户评价区
- 底部固定栏：❤️ 想去 / 🧭 导航 / ➕ 在这里发布鱼获

### 7.2 鱼获详情页

**元素清单**
- 顶部大图轮播（可全屏看图）
- 用户信息条
- 鱼种 + 重量 + 长度的数据条

### 8.1 用 AI 出图的工作流建议

### 8.2 优先出图顺序

## 9. 待你确认 & 调整的设计点

> 看完后告诉我哪些要改

1. **整体风格定调**：默认了"自然清新+工具感"，要不要换成"硬核工具"（深色专业向）或"年轻潮玩"（鲜艳卡通向）？
2. **主色 #2D8F87 墨绿**：可以接受，还是要换成更亮的蓝绿？
3. **TabBar 中间"+"**：是凸起按钮（小红书风）还是平的（小程序常见风）？
4. **首页默认视图**：默认地图模式，还是默认列表模式？很多老钓友其实更习惯列表。
5. **我的页头部**：渐变+背景图够不够，还是要更朴素？
6. **是否要在 v1 加"鱼库成就墙"**：这个增加开发量，但对中年男性用户的"收集欲"满足感很强。
